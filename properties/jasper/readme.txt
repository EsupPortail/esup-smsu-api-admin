/*****************************/
/  Description des fichiers   /
/*****************************/

 > accounts_report.jrxml :	
Ce fichier est le modèle du report construit à l'aide de iReport.
C'est ce fichier qu'il faut modifier pour faire évoluer le report.

 > accounts_report.jasper :
Ce fichier est le modèle compilé du fichier précédent (compilé avec iReport).
C'est ce fichier que l'application smsu-api-admin utilise.
